# TTAM Toolkit

一个给 TikTok Ads 运营用的纯前端小工具，双击即用，无需安装。

---

## 功能介绍

### 1. TTAM 链接生成

#### Account Level
- 页面顶部有 **账号类型 Tab**（如 RTA / Pangle / OM Branding），点击切换
- 每种类型下展示对应的市场卡片（BR / ID / PH / TH / VN / MY / SG / TW / AR 等）
- 每张卡片显示 **Market** 和 **Account Name**
- 卡片底部有两个按钮：
  - **Account** → 直接打开该账号的 TikTok Ads Dashboard
  - **Camp** → 自动滚动到下方 Camp Level，并帮你预选好该账号
- 支持 **其他账号** 快速输入：输入任意 Adv ID，一键 Dashboard 或一键 Camp

#### Camp Level
- 通过 **级联选择器** 选择账号：先选类型 → 再选 Market → 最后选具体账号
- 支持粘贴 Camp ID（空格、换行、英文逗号、中文逗号混输均可）
- 选择日期范围，点击 **生成链接**
- 支持一键 **打开链接** 或 **复制链接**

### 2. 格式转换

日常数据清洗小帮手：
- 把各种分隔符混输的 Camp ID 转成标准格式
- 支持 **每行一个** 或 **英文逗号分隔**
- 支持 **去重**
- **防抖实时转换**：输入时自动处理，无需频繁点击按钮

## 如何使用

直接用浏览器打开 `index.html` 即可：

```
双击 index.html，或把文件拖到浏览器里
```

> 这是一个纯前端网页，不需要服务器，也不需要联网（除了生成的 TTAM 链接本身需要网络）。

## 如何修改/更新账号

**所有账号信息都存在 `app.js` 一个文件里。**

### 步骤：
1. 用任意文本编辑器打开 `app.js`
2. 找到最顶部带有 `【账号底表】` 注释的区域
3. 修改或新增账号

例如新增一个市场：

```javascript
  { type: 'RTA', market: '新市场', name: '账号名字', advId: '你的账号ID' },
```

### 字段说明：
| 字段 | 含义 | 示例 |
|------|------|------|
| `type` | 账号类型 | `'RTA'`、`'Pangle'`、`'OM Branding'` |
| `market` | 市场代码 | `'BR'`、`'ID'`、`'AR'` 等 |
| `name` | 账号名称 | `'Shopee BR - RTA'` |
| `advId` | 广告账号 ID | `'7369540846363983888'` |

### 分享给同事：
直接把整个文件夹发给同事，同事只需要改 `app.js` 顶部的那几行账号 ID，就能用自己的账号运行。

## 注意事项

- 日期默认显示最近 7 天，可手动调整
- Camp ID 输入支持混合格式，系统会自动清洗
- 格式转换区的"去重"默认开启
- 新增 `type` 时，Account Level 的 Tab 和 Camp Level 的级联选择器都会**自动识别**，无需改其他地方
